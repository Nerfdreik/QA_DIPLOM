# Автоматизация тестирования книжного магазина Labirint.ru

## Описание проекта

Этот проект представляет собой комплексную систему автоматизации тестирования для книжного магазина Labirint.ru. Проект включает как UI-тесты (с использованием Selenium WebDriver), так и API-тесты (с использованием библиотеки requests).

### Основные возможности:
- **5 UI тестов** для проверки функциональности интерфейса
- **5 API тестов** для проверки работы поиска и других функций
- Поддержка **Allure отчетов** для наглядной визуализации результатов
- Разделение тестов по маркерам для гибкого запуска
- Конфигурация через переменные окружения
- Логирование и создание скриншотов при падении тестов

Запуск тестов
1. Запуск всех тестов
bash
pytest
2. Запуск только UI тестов
bash
pytest -m ui
3. Запуск только API тестов
bash
pytest -m api
4. Запуск с генерацией Allure отчета
bash
# Запуск тестов с сохранением результатов
pytest --alluredir=allure-results

# Генерация и открытие отчета
allure serve allure-results

# Или генерация статического отчета
allure generate allure-results -o allure-report
5. Запуск с дополнительными опциями
bash
# С подробным выводом
pytest -v

# С логированием
pytest -v --log-cli-level=INFO

# С перезапуском упавших тестов

pytest --reruns 2 --reruns-delay 1
